SUMÁRIO EXECUTIVO

Durante esta semana, a equipe deu continuidade ao desenvolvimento do Sistema de Gestão de Estoque da START Reciclagem, empresa voltada à reciclagem de materiais e produção de móveis sob encomenda.

O sistema está sendo desenvolvido utilizando Python e Flask no backend, HTML, CSS e JavaScript no frontend, além de SQLite como banco de dados local para armazenamento das informações.

Foram implementadas melhorias significativas na interface do sistema, incluindo dashboard gerencial com indicadores de estoque, gráficos para visualização de dados, sistema de autenticação de usuários, geração de SKU para produtos, controle de movimentações de estoque, exportação de relatórios em PDF, Excel e CSV, além de funcionalidades de impressão.

Também foi estruturado o banco de dados SQLite, contendo tabelas para armazenamento de usuários, produtos e movimentações, garantindo a persistência das informações e permitindo o acompanhamento das entradas e saídas de materiais.

Além disso, foram iniciados recursos voltados à experiência do usuário, como perfil de usuário, notificações, relógio em tempo real e modo claro/escuro.

Como próximos passos, a equipe pretende aprimorar o dashboard, tornando os gráficos mais interativos e personalizáveis, além de implementar filtros temporais por semana, mês, semestre e ano. Também serão realizadas correções no sistema de alternância entre modo claro e escuro, que ainda não está funcionando perfeitamente em todas as telas.

Está prevista a integração com o Supabase para substituição ou complementação do banco de dados local SQLite. A integração ainda não foi realizada porque o projeto encontra-se em fase de ajustes e correções. Neste momento, a prioridade é corrigir problemas identificados na interface, melhorar o dashboard e implementar novas funcionalidades. Após a estabilização do sistema, será iniciada a integração com o Supabase para permitir armazenamento em nuvem, sincronização de dados e maior escalabilidade da aplicação.

Atualmente, o sistema encontra-se funcional para demonstração e testes, atendendo aos principais requisitos definidos para esta etapa do projeto.
